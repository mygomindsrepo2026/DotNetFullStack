#  Inheritance in OOPS

---

##  Introduction

**Inheritance** allows one class to **inherit properties and methods** from another class.

👉 Promotes **code reusability**

---

##  Real-Time Example

###  Vehicle → Car

* Vehicle has Start()
* Car inherits and uses it

---

##  Example (C#)

```csharp
class Vehicle
{
    public void Start()
    {
        Console.WriteLine("Vehicle started");
    }
}

class Car : Vehicle
{
}
```

---

## 🔁 Types of Inheritance

* Single Inheritance
* Multilevel Inheritance
* Hierarchical Inheritance

---

##  Key Points

* Uses `:` symbol
* Reuses existing code
* Reduces duplication

---

##  Advantages

* ✅ Code reusability
* ✅ Faster development
* ✅ Easy maintenance

---

##  Conclusion

Inheritance helps in:

* Reusing existing functionality
* Building relationships between classes
