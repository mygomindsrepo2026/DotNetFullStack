# Object-Oriented Programming (OOPS)

---

##  What is OOPS?

**OOPS (Object-Oriented Programming System)** is a programming approach based on the concept of **objects**.
Objects contain **data (variables)** and **methods (functions)** that operate on the data.

It helps in organizing code in a structured and reusable way.
