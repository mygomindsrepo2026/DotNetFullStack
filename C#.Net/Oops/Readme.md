## 📌 What is OOPS?

**OOPS (Object-Oriented Programming System)** is a programming paradigm based on the concept of **objects**.

💡 Objects contain:
- **Data (Variables)**
- **Methods (Functions)**

👉 It helps in building:
- Scalable applications  
- Reusable code  
- Clean architecture
